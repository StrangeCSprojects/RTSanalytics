to run the UI 
you need to install Node.js
https://nodejs.org/en/learn/getting-started/how-to-install-nodejs

this will install npm, with npm you will want to run this command to start a project
npx create-react-app ui
cd ui

then copy the src file and paste it.

to run and edit, do 
npm start
 -this will create a local server and start the website/web app
 -when changes are saved, this will be updated live
 -ctrl + c on cmd to stop the server

for builds, run 
npm run build

npm install -g serve

then 

serve -s build

this should act as a server running the react file.
